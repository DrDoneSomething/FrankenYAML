This is a list of Settings integrations.

They are basically a mix between dictionary and list definitions that have stay within the integration file.

They are barely changed at all when parsed, except !include* paths are corrected to reflect their relative locations to the integration yaml file, and Nested integrations are added (eg. customize).

If there is a list or dictionary within a setting you wish to have in a separate file, eg dictionary "customize:", add it to the appropriate Nested Dictionary or Nested List configuration below.
